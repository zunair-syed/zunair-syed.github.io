Light Bulb Motion Controller

Summary

This project was completed for HackTheNorth 2016. It basically controls the Phillip Hue lightbulb using the LeapMotion as a controller. This allows the user to change the colour and saturation of the bulb using only their hands.

Technical Description

Used LeapMotionSDK and the Phillips Hue API in JavaScript. LeapMotion is a device that detects hand gestures above the leapmotion device. The Phillips Hue Bulb can change its colour, brightness and saturation given a response from its API.
