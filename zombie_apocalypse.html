Zombie Apocalypse

"Zombie Apocalypse" is an amusing 2D Java Applet based game, which depicts the famous character "Megaman" facing off zombies. This game utilizes numerous sets of sprites, double buffering, and the JApplet API. This game was made on the Eclipse IDE, and this game can be run on any browser since this game is made on the JApplet API.
